### 接口说明

URL:http://N1的IP:8080/v1/keyevent  (发送按钮指令)
方式：POST
参数内容：

{"keycode":按键代码,"longclick":false}
按键代码列表：

- 上:19
- 下:20
- 左:21
- 右:22
- 返回:4
- 音量加:24
- 音量减:25
- 主界面:3
- 菜单:82
- 确认键:23
- 电源:26

URL: http://N1的IP:8080/v1/action (打开设置界面)
方式：POS
参数内容：

{"action":"setting"}
